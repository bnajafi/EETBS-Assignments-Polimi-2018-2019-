def lag_feature(df,column_name,lag_start,lag_end,lag_interval):
    for i in range(lag_start,lag_end+1,lag_interval):
        new_column_name=column_name+"-"+str(i)+"hr"
        print new_column_name
        df[new_column_name]=df[column_name].shift(i)
        df.dropna(inplace=True)
    return df

Machine learning¶

In the second part I want to predict the consumptions of the building . To do this we can use the "machine learning" strategy, so we need to create a model for the computer so that it can predict the consumption with only the informations that we have.¶

In order to do our prediction in machine learning, we need to create two sets of our datas:training and test(20% for test) ,we will fit our model on the train data in order to make predictions on test datas. The "training" set contains a known output and the model learns on this data in order to be generalized to other data later on. The "test" dataset will be used to test our model’s prediction on this set.¶

So,for first we need to import "train_test_split" in order to split the datas in training and test data¶

In the end we use another method. It's called online learning and in this way we use only a part of datas,is an approach that ingests data one observation at a time.¶

In [87]:
DF_onlineConsumptionPrediction=pd.DataFrame(index=DF_mod.index)

We have to choose a period of training, that in our case is the month of March. Then we run a "while" cycle. This predict the first hour after the training period, so the first hour of April and then we update the period of training with the real value.¶

We choose a dataframe for the period of training called "period_of_training".¶

There are two chances for the while cycle:a cycle with linear regression method and another one with random forest¶

We can use random forest because is more accurate. Infact it's possible to see this in the final result.¶

In [67]:
period_of_training=pd.Timedelta(30,unit="d")

FirstTimeStamp_measured=DF_mod.index[0] #first hour of considered period
LastTimeStamp_measured=DF_mod.index[-1]  #last hour of considered period

FirstTimeStamp_toPredict=FirstTimeStamp_measured+period_of_training #the first next hour after period of training

training_startTimeStamp=FirstTimeStamp_measured 
training_endTimeStamp=FirstTimeStamp_toPredict 
timeStamp_toPredict=FirstTimeStamp_toPredict   #first hour predicted
DF_onlineConsumptionPrediction=DF_onlineConsumptionPrediction.truncate(before=training_endTimeStamp)
#we create a cycle that allows us to update 
while (timeStamp_toPredict< LastTimeStamp_measured):
    
    #DF_feature_train=DF_features.loc[training_startTimeStamp:training_endTimeStamp]
    DF_feature_train=DF_features.truncate(before=training_startTimeStamp,after=training_endTimeStamp)
    DF_target_train=DF_target.truncate(before=training_startTimeStamp,after=training_endTimeStamp)

    DF_feature_test=DF_features.loc[timeStamp_toPredict].values.reshape(1,-1)
    DF_target_test=DF_target.loc[timeStamp_toPredict]
    
    reg_RF.fit(DF_feature_train,DF_target_train)  #reg_RF instead of linear_reg
    predicted_Consumption=reg_RF.predict(DF_feature_test)
    DF_onlineConsumptionPrediction.loc[timeStamp_toPredict,"Predicted"]=predicted_Consumption
    DF_onlineConsumptionPrediction.loc[timeStamp_toPredict,"Real"]=DF_target_test
    
    
    timeStamp_toPredict=timeStamp_toPredict+pd.Timedelta(1,unit="h")
    training_endTimeStamp=training_endTimeStamp+pd.Timedelta(1,unit="h")
    training_startTimeStamp=training_startTimeStamp+pd.Timedelta(1,unit="h")
